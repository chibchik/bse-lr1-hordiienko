#Мій перший проєкт

Калькулятор - це програма, яка дозволяє користувачеві виконувати арифметичні операції, такі як додавання, віднімання, множення та ділення. У цьому калькуляторі користувач може вводити числа та вибирати операцію, яку він хоче виконати. Програма обчислює результат операції та виводять його на екран.

Ось приклад використання калькулятора:

Виберіть операцію:
1. Сума
2. Віднімання
3. Множення
4. Ділення
Ваш вибір: 1
Введіть перше число: 5
Введіть друге число: 3
5 + 3 = 8

Такий калькулятор може бути корисним для швидкого обчислення арифметичних операцій.
