#Мій перший проєкт

Калькулятор - це програма, яка дозволяє користувачеві виконувати різноманітні арифметичні та математичні операції. У цьому калькуляторі користувач може вводити числа та вибирати операцію, яку він хоче виконати. Програма обчислює результат операції та виводить його на екран.

## Підтримувані операції:

1. **Сума** - додавання двох чисел
2. **Віднімання** - віднімання другого числа від першого
3. **Множення** - множення двох чисел
4. **Ділення** - ділення першого числа на друге (з перевіркою на нуль)
5. **Піднесення до степені** - піднесення першого числа на степінь другого числа
6. **Квадратний корінь** - обчислення квадратного кореня першого числа
7. **Відсоток від числа** - обчислення вказаного відсотка від числа

## Приклад використання:

```
Виберіть операцію:
1. Сума
2. Віднімання
3. Множення
4. Ділення
5. Піднесення до степені
6. Добуток квадратного кореня
7. Відсоток від числа
Ваш вибір: 1
Введіть перше число: 5
Введіть друге число: 3
5 + 3 = 8
```

Такий калькулятор може бути корисним для швидкого обчислення різних арифметичних та математичних операцій.
